Notes

Every script that has been edited by Tsalkapone contains thorough guides, regarding each script's functions and customizable variables, that can be found wherever is needed ingame. All you have to do is click on any you may notice.

Most of the scripts edited by Tsalkapone contain more than one supported language. You can find the supported languages of every script by clicking the button Open language selection when activating the script. You can change the selected language by clicking on the desired one.

Whenever a script hosted by Tsalkapone is updated you will get notified about its new version and changes log ingame when you activate it.

All the saved values of the scripts are stored in the localStorage, cache or a cookie of your browser. Changing browser or deleting your cookies will remove any saved value from any script.

The table displayed below supports a large variety of filters, columns and information that can be adjusted accordingly. The Column filter settings are automatically saved in your browser in every change. You may combine various filters to get the result you want. The Search filter cancels every other filter setting.

The scripts on the table are ordered based on the Date Entry of each script. You may change the order by clicking on the heading of each visible column.

By moving your cursor over any column, row, filter or image you will get all the info you need about its meaning and usage.

Translation Instructions

Every script that has been edited by Tsalkapone supports more than one language. You may select the desired language through the settings table which is embedded in a script. Scripts that don't support settings table will use the selected language of other scripts with similar functions. Click on any script to view its description for more info regarding its language selection.

The Trans column on the table below displays an icon-link that opens each script's translations file on a new tab. On the top of each file between /* and */ tags you may read more instructions regarding each translation file. The parts (words or sentences) of each translation file that are to be translated are the ones displayed between " and ". Ignore every character, word or sentence inside these characters: < >. Also, try to keep the punctuations in each variable as they are.

There are some scripts which contain specific parts that have to be translated to their relevant ones if you want to use a script in any other server apart from .uk, .us and .net. The variable tw in a translation file contains all these "must" parts. If you just need to use a script in any other server besides the ones already mentioned just make sure to translate these parts.

Some scripts contain large translation files. Some of them are included in the translation file with the tag main and some of them with the tag advanced. The 1st category contains parts that are displayed on the screen. The 2nd category contains parts that are included in additional infos pop-ups.

After you complete a translation, send a message to tsalkapone@hotmail.com including in the mail's title the following infos:
1. The script's name.
2. The translated language.
3. The translator's name (pick anything you want to be displayed on the table below).
i.e. Commander Script - French (MrIncognito).

Technical Notes & Bug Reports

LocalStorage & cookies
Cookies and local storage serve different purposes. Cookies are primarily for reading server-side, local storage can only be read client-side. Unlike cookies, LocalStorage is more secure and large amounts of data can be stored locally, without affecting website performance. Most of the scripts hosted by Tsalkapone use LocalStorage to save the variables' values. Cookies are used only for scripts running from the rally point page. Take into consideration that you may clear any LocalStorage or cookie used in a browser. Insert the following urls, for each browser, in your address bar to access the menu of the saved cookies and data per website:
Opera: opera://settings/cookies
Mozilla: about:preferences#privacy
Opera: chrome://settings/cookies
Once you open the menu click on the desired website (e.g. tribalwars.net) and select the saved data you would like to delete.

Pop-up windows
Some of the scripts may trigger one or more pop-up windows. By default, your browser will forbid the opening of the pop-up and a notification of restriction will be displayed on your address bar. Make sure that you have granted permission for pop-up windows for tribalwars website. You may change your pop-up setting for the current website from your address bar. If a pop-up is blocked you will get notified with a Pop-ups blocked. Click on it and click on Always show pop-ups from [site].
Alternatively access the settings menu of your browser and follow these instrunctions for each browser.

Quick bar icons
Beware to load valid image-urls on your quick bar. If by any case, a loaded url is no longer valid you may notice lags for more than 5 seconds in every page refreshing. Better load the icons provided by the game for this kind of purpose.

Bug reports
Some of the scripts hosted by Tsalkapone provide you with reports you can use to ask for help if you encounter any bug or malfunction in the script's function(-s). Search for them in the bottom of the page. Furthermore, provide as many information as you can regarding how, where and when the bug/malfunction occured. Alternatively, press F12 on your keyboard and copy every mark noted as Error in your console log. Make sure you send all these details to Tsalkapone or any other scripter you trust.
